Personal Portfolio

For my independent project in SMAD 408, I chose to build my personal portfolio site from the ground up. I learned a lot of skills completing this project, and although changes still need to be made, I'm proud of the final result. I sketched a logo and came up with a visual style for the site, which I then developed using GitHub and Adobe Dreamweaver. The final site uses a domain name I purchased, which is hosted through Netlify. Netlify lets me connect to my GitHub repository for the site, making changes quick and easy.

Skills

As part of SMAD 408, there were multiple opportunities for learning new skills. These are the LinkedIn Learning Certificates I earned as part of the course.

In order to complete my independent project (my personal portfolio website) and the capstone project (Story Analyzer), I learned how to use GitHub, both the desktop and in-browser versions. For the portfolio website, I watched a quick course on what goes into a good online portfolio, and I also took the Bootstrap 5 essential training. Learning how to use Bootstrap also helped a lot with the capstone project, and this visual diary! Out of all the skills I learned in this class, Bootstrap 5 was probably the most beneficial. It was also important that I learn the Agile project management methodology, so I took a foundations course in order to better work with my CIS team on the capstone project.

It's important to note that I also picked up some other skills throughout the coursework, such as:

  • Working within a multidisciplinary team
  • Buying a domain name
  • Hosting a website
  • Working with quick project turnaround and deadlines
  • Working with a client
  • Using feedback to improve future project iterations

Sketchbook Reflection

Throughout SMAD 408, we completed multiple "Sketchbook Reflection" assignments, where we would use sketches to learn design concepts, reflect on previous work, and begin thinking through the capstone project. We posted our work to discussion boards, where we commented on other classmates' work and thought about our own. Here you can see some of the sketches I completed throughout the semester. As you can see, my sketches improved the more I completed. If you look at my sketches contained within other sections of this visual diary, they also improve along with these sketchbook reflections. Part of this assignment was learning how to put effort into every sketch, and really caring about organization and each step of the process. I'm proud of where my sketching has ended up.
